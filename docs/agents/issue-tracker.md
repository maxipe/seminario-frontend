# Issue Tracker: GitHub

Issues for this project live in GitHub Issues.

## CLI

Use the `gh` CLI for all issue operations:
- Create: `gh issue create`
- List: `gh issue list`
- View: `gh issue view <number>`
- Edit: `gh issue edit <number>`
- Close: `gh issue close <number>`

## Conventions

- Always add labels when creating issues.
- Link related PRs to issues via `gh pr create --body "Closes #<number>"`.
- Use the milestone field to group issues by release when applicable.
